# Api Repositórios Github

Essa é uma api que pode ser usada por qualquer pessoa para listar os repositórios existentes do github para um determinado usuário.

## Tecnologias usadas
* Spring-Boot
* Spring-security-jwt
* Java 1.8 ou superior
* Docker

## Ferramentas
* Maven 4.0
* IDE - Intellij Ultimate
